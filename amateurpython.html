Cours Python pour Amateurs

Bienvenue dans notre cours Python pour débutants. Dans ce cours, vous apprendrez les bases de la programmation Python, un langage convivial et puissant. Que vous soyez totalement novice en programmation ou que vous souhaitiez renforcer vos compétences, ce cours est conçu pour vous.

Contenu du Cours

Ce cours est divisé en plusieurs sections, chacune couvrant un aspect clé de la programmation Python. Voici un aperçu du contenu :

Section 1 : Introduction à Python

Dans cette section, vous découvrirez ce qu'est Python, son histoire et ses utilisations courantes. Nous vous montrerons comment installer Python sur votre ordinateur et comment écrire et exécuter votre premier programme Python.


# Votre premier programme Python
print("Bonjour, Python!")

Section 2 : Variables et Opérations de Base

Dans cette section, nous aborderons la création de variables pour stocker des données, ainsi que les opérations de base sur les nombres et les chaînes de caractères.


# Déclaration de variables
nom = "Alice"
age = 25

# Opérations de base

Section 3 : Structures de Contrôle

Vous apprendrez comment prendre des décisions en utilisant des déclarations conditionnelles (if) et comment répéter des actions avec des boucles (for et while). Ces concepts sont essentiels pour contrôler le flux de votre programme.


# Déclaration conditionnelle
if age >= 18:
    print("Vous êtes majeur.")
else:
    print("Vous êtes mineur.")

# Boucle for
for i in range(5):
    print(i)

Section 7 : Fonctions Avancées

Dans cette section, nous explorerons les fonctions avancées en Python, telles que les fonctions récursives et les fonctions anonymes (lambda).


# Fonction récursive
def factorielle(n):
    if n == 0:
        return 1
    else:
        return n * factorielle(n - 1)

# Fonction lambda
carre = lambda x: x ** 2

Section 14 : Programmation Orientée Objet

Vous découvrirez les concepts de la programmation orientée objet (POO) en Python et comment créer des classes et des objets.


# Programmation orientée objet
class Personne:
    def __init__(self, nom, âge):
        self.nom = nom
        self.âge = âge

    def présenter(self):
        print(f"Je m'appelle {self.nom} et j'ai {self.âge} ans.")

# Créer un objet
alice = Personne("Alice", 30)
alice.présenter()

Section 19 : Apprentissage Automatique avec Scikit-Learn

Dans cette section, vous découvrirez comment créer des modèles d'apprentissage automatique en utilisant Scikit-Learn.


# Apprentissage automatique avec Scikit-Learn
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier

# Charger un jeu de données
iris = load_iris()
X, y = iris.data, iris.target

# Diviser les données en ensembles d'entraînement et de test
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# Créer un modèle de forêt aléatoire
modèle = RandomForestClassifier(n_estimators=100)
modèle.fit(X_train, y_train)
précision = modèle.score(X_test, y_test)
